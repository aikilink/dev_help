My Journey and Recommendations

I began really digging into the web-dev world, when I started this course at Udemy: zero-to-mastery. I would highly recommend this course to anyone who feels the need for some structured guidance, and some great videos and resources!

Here are some resources that I found most helpful.

General

  • W3 Schools is one of the best resources out there for web development. I highly recommend visiting it frequently when you have questions.

  • Sololearn has an app that comes in handy as an extra resource. You can select on option to make the courses available offline, which comes in handy.

  • Unsplash is an awesome resource for free images that you can use for your websites or personal use. Crediting the artist isn't required, but is recommended.

  • Undraw is similar to Unsplash, but for SVG files.

  • Zero-To-Mastery resource list.This was made by my fellow students. It is a very thorough list of guides and resources for the aspiring web developer!

  • Develop a Job is a good one-stop location to get started on your journey!

HTML

  • W3 Schools HTML5.

CSS

  • W3 Schools CSS.

  • Flexbox Froggy is a fun game to play for learning to use CSS Flexbox.

  • Grid Garden is a great little game for learning CSS Grid concepts.
